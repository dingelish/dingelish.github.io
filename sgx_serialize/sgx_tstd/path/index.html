Module sgx_tstd::path [−] [src]

Cross-platform path manipulation.

This module provides two types, [PathBuf] and [Path]Path (akin to [String] and [str]), for working with paths abstractly. These types are thin wrappers around [OsString] and [OsStr] respectively, meaning that they work directly on strings according to the local platform's path syntax.

Paths can be parsed into [Component]s by iterating over the structure returned by the [components] method on [Path]. [Component]s roughly correspond to the substrings between path separators (/ or \). You can reconstruct an equivalent path from components with the [push] method on [PathBuf]; note that the paths may differ syntactically by the normalization described in the documentation for the [components] method.
